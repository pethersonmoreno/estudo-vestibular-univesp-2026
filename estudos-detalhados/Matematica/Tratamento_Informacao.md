O tratamento da informação é uma área da matemática que lida com a coleta, organização, análise e interpretação de dados. É fundamental para a compreensão do mundo contemporâneo e é um tema de alta frequência em vestibulares, abrangendo desde estatística básica até noções de algoritmos.

## 9.1. Gráficos, Tabelas e Planilhas

### Tabelas

Organizam dados em linhas e colunas, facilitando a visualização e a comparação de informações.
-   **Elementos**: Título, cabeçalhos de coluna, cabeçalhos de linha, dados, fonte.

### Gráficos

Representações visuais de dados, que facilitam a identificação de padrões, tendências e comparações.
-   **Gráfico de Barras**: Usado para comparar quantidades entre categorias distintas.
-   **Gráfico de Setores (Pizza)**: Mostra a proporção de cada parte em relação a um todo (geralmente em porcentagem).
-   **Gráfico de Linhas**: Ideal para mostrar a evolução de dados ao longo do tempo (séries temporais).
-   **Histograma**: Semelhante ao gráfico de barras, mas as barras são adjacentes e representam a distribuição de frequência de dados contínuos.
-   **Gráfico de Dispersão**: Mostra a relação entre duas variáveis numéricas, revelando possíveis correlações.

### Planilhas Eletrônicas (Conceito)

Ferramentas como Excel ou Google Sheets são usadas para organizar, calcular e analisar dados de forma eficiente. Permitem realizar cálculos complexos, criar gráficos e gerenciar grandes volumes de informação.

## 9.2. Medidas de Tendência Central e de Dispersão

### Medidas de Tendência Central

Indicam o valor que melhor representa o centro de um conjunto de dados.
-   **Média Aritmética**: Soma de todos os valores dividida pelo número de valores. ($\bar{x} = \frac{\sum x_i}{n}$).
-   **Mediana**: Valor central de um conjunto de dados ordenado. Se o número de dados é ímpar, é o valor do meio; se é par, é a média dos dois valores centrais.
-   **Moda**: Valor que mais aparece em um conjunto de dados (maior frequência).

### Medidas de Dispersão

Indicam o quanto os dados estão espalhados ou variam em relação à média.
-   **Amplitude**: Diferença entre o maior e o menor valor do conjunto de dados.
-   **Variância ($\sigma^2$)**: Média dos quadrados das diferenças entre cada valor e a média aritmética. Indica o quão "longe" os valores estão da média.
-   **Desvio Padrão ($\sigma$)**: Raiz quadrada da variância. É a medida de dispersão mais utilizada, na mesma unidade de medida dos dados.

## 9.3. Representação, Interpretação e Resolução de Problemas Envolvendo Algoritmos. Fluxograma. Conceitos Básicos de Linguagem de Programação

### Algoritmos

Um **algoritmo** é uma sequência finita de passos claros, lógicos e não ambíguos para resolver um problema ou executar uma tarefa. É a base da programação de computadores.

**Características de um bom algoritmo:**
-   **Finito**: Termina em um número finito de passos.
-   **Preciso**: Cada passo é bem definido.
-   **Definido**: Produz o mesmo resultado para os mesmos dados de entrada.
-   **Eficaz**: Os passos devem ser realizáveis.

*Exemplo de algoritmo para somar dois números*: 
1.  Leia o primeiro número (A).
2.  Leia o segundo número (B).
3.  Calcule a soma: S = A + B.
4.  Escreva o resultado (S).

### Fluxograma

Um **fluxograma** é uma representação gráfica de um algoritmo, utilizando símbolos padronizados para representar as operações e a sequência de passos. É uma ferramenta visual para planejar e documentar processos.

**Principais Símbolos:**
-   **Retângulo arredondado**: Início/Fim.
-   **Paralelogramo**: Entrada/Saída de dados.
-   **Retângulo**: Processamento (cálculos, atribuições).
-   **Losango**: Decisão (condicionais, "Sim" ou "Não").
-   **Setas**: Conectores (indicam a sequência do fluxo).

### Conceitos Básicos de Linguagem de Programação

Embora não seja uma aula de programação, as noções básicas são importantes:
-   **Variáveis**: Espaços na memória para armazenar dados (números, textos).
-   **Comandos de Entrada/Saída**: Funções para ler dados do usuário (entrada) e exibir resultados (saída).
-   **Estruturas Condicionais (SE/ENTÃO/SENÃO - if/else)**: Permitem que o algoritmo execute diferentes blocos de código dependendo de uma condição.
-   **Estruturas de Repetição (ENQUANTO/PARA - while/for)**: Permitem que um bloco de código seja executado repetidamente.
-   **Funções/Sub-rotinas**: Blocos de código que realizam uma tarefa específica e podem ser reutilizados.

## Exercícios Estilo Vunesp

**Questão 1 (Vunesp Adaptada):**
Em uma pesquisa de salários, foram coletados os seguintes valores em milhares de reais: 2, 3, 5, 5, 8, 12, 15. Qual é a média, a mediana e a moda desses salários, respectivamente?

a) 7, 5, 5
b) 5, 7, 5
c) 7, 5, 8
d) 8, 7, 5
e) 5, 5, 7

**Resolução:**
Dados: (2, 3, 5, 5, 8, 12, 15)
-   **Média**: Soma dos valores / Quantidade de valores
    Soma = $2+3+5+5+8+12+15 = 50$
    Quantidade = 7
    Média = $50 / 7 \approx 7.14$. Arredondando para o mais próximo na alternativa, 7.
-   **Mediana**: Os dados já estão ordenados. Há 7 valores, então o valor central é o 4º termo.
    Mediana = 5.
-   **Moda**: O valor que mais se repete.
    Moda = 5.

Considerando as alternativas e a média aproximada, a que melhor se encaixa é (7, 5, 5).

**Alternativa Correta: a)**

**Questão 2 (Vunesp Adaptada):**
Um fluxograma é uma ferramenta para representar algoritmos de forma visual. Qual dos símbolos abaixo é utilizado para representar uma **decisão** (verificação de uma condição) em um fluxograma?

a) Retângulo
b) Paralelogramo
c) Círculo
d) Losango
e) Retângulo arredondado

**Resolução:**
Vamos rever os símbolos do fluxograma:
-   Retângulo: Processamento (cálculos, atribuições).
-   Paralelogramo: Entrada/Saída de dados.
-   Círculo: Não é um símbolo padrão para decisão (geralmente usado para conectores em outras páginas).
-   Losango: Correto. É o símbolo universalmente aceito para representar uma decisão ou uma verificação condicional em um fluxograma.
-   Retângulo arredondado: Início/Fim do algoritmo.

**Alternativa Correta: d)**
